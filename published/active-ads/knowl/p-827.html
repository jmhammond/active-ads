Suppose a two bit message is encoded into a five bit message using the function \(e(b_{1}b_{2})=b_{1}b_{1}(b_{1}+_2 b_{2})b_{2}b_{2}\text{.}\) Determine a matrix \(P\) such that if a two bit message, \(b = b_{1}b_{2}\text{,}\) is encoded and \(e(b)\) transmitted, then any single bit error in the received string \(c=c_{1}c_{2}c_{3}c_{4}c_{5}\) can be identified by whether \(c P\) is the zero vector or not.
