  2. Recall that if \([G;*]\) is a finite group and \(a \in G\text{,}\) the cyclic subgroup generated by \(a\) is \(\langle a \rangle=\{a, a*a, a*a*a, \dots\}\text{,}\) which is a finite set since you eventually get to the identity and can stop listing elements. For example, if the group is \(\mathbb{Z}_{6}\text{,}\) there are four different cyclic subgroups: \(\langle 0 \rangle\text{,}\) \(\langle 1 \rangle\text{,}\) \(\langle 2 \rangle\text{,}\) and \(\langle 3 \rangle\text{.}\) The other two elements of \(\mathbb{Z}_{6}\text{,}\) 4 and 5 generate subgroups that are equal to the subgroups generated by 2 and 1, respectively. The Hasse diagram for this set of subgroups with \(\subseteq\) is Figure 40.2.1
    1. List the distinct cyclic subgroups generated by elements of \(\mathbb{Z}_{12}\text{.}\) The set of subgroups you’ve found is a lattice with the partial ordering \(\subseteq\text{.}\) Draw the Hasse diagram for this poset.
    2. Do the same with the cyclic subgroups generated by elements of \(\mathbb{Z}_{16}\text{.}\)
    3. Do the same with the cyclic subgroups generated by elements of \(\mathbb{Z}_{30}\text{.}\)
    4. Any general observations from the three cases?
