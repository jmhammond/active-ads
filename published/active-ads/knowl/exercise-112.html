Exercise 1.
Let \(A = \{1, 2, 3, 4\}\text{.}\) Define \(f:A\rightarrow A\) by \(f(1) = 2\text{,}\) \(f(2) = 3\text{,}\) \(f(3) = 4\text{,}\) and \(f(4) = 1\text{.}\) Find \(f^2\text{,}\) \(f^3\text{,}\) \(f^4\) and \(f^{-1}\text{.}\) You can describe each of these functions as being equal to a previous one, or in the same manner as \(f\) was originally described.
