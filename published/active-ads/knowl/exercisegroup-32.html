Exercise Group 18.2.1–4.

1.
Define functions on the positive integers, \(\mathbb{P}\text{,}\) if they exist, that have the properties specified below.
  1. A function that is one-to-one and onto.
  2. A function that is neither one-to-one nor onto.
  3. A function that is one-to-one but not onto.
  4. A function that is onto but not one-to-one.
2.
Prove that in a room with \(n\) people, \(n \geq 2\text{,}\) at least two people know exactly the same number of people. Assume knowing is a symmetric relation: If Paul knows Pat, then Pat knows Paul.
3.
Infinite Acres Spa and Math Camp has an infinite number of single occupancy rooms, numbered with each positive integer. You are the night manager. The spa is fully booked for the weekend and all rooms are occupied. A bus arrives late Friday night. You find that the manager has booked an additional infinite busload of customers, with confirmation codes numbered \(1, 2, 3, \dots\text{.}\) Can you accomodate the new arrivals?
4.
Prove that the set of finite strings of 0’s and 1’s is countable.
