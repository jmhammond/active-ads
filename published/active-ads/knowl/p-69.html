Let \(X = \{n \in \mathbb{N} \mid 10 \leq n \lt 20\}\text{.}\) Find examples of sets with the properties below and very briefly explain why your examples work.
  1. A set \(A \subseteq \mathbb{N}\) with \(\lvert A \rvert = 10\) such that \(X - A = \{10, 12, 14\}\text{.}\)
  2. A set \(B \in \mathcal{P}(X)\) with \(\lvert B\rvert = 5\text{.}\)
  3. A set \(C \subseteq \mathcal{P}(X)\) with \(\lvert C\rvert = 5\text{.}\)
  4. A set \(D \subseteq X \times X\) with \(\lvert D\rvert = 5\text{.}\)
  5. A set \(E \subseteq X\) such that \(\lvert E\rvert \in E\text{.}\)
