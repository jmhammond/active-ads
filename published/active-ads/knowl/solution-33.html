Solution 34.2.3.1.
Let \(H\) be the set of distinct powers of \(a\text{.}\)  This is a finite set since it’s a subset of a finite set.  Pick any element \(a\) of \(G\) that’s not in \(H\text{.}\)  The function \(f_a:H \rightarrow G\) from the last problem of the week is a one-to-one function (this implied by the cancellation laws).  The range of \(f_a\text{,}\)  \(f_a(H)\text{,}\) has the same cardinality as \(H\text{.}\)  Keep repeating the process, at each stage selecting an element of \(G\) that’s not appeared in \(H\) or any of the ranges of functions from previous steps.   This process has to end in a finite number of steps and produces a partition of \(G\) into subsets all of the same cardinality as \(H\text{.}\)
