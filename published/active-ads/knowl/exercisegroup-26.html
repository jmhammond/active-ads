Exercise Group 15.2.1–4.

1.
Let \(S\) be the set of “spaces” in the floor of your classroom. Draw a digraph of the relation \(c\text{,}\) where \(s_1 c s_2\) if and only if \(s_1\) is connected to \(s_2\) with at least one doorway.
2.
Given \(s\) and \(t\text{,}\) relations on \(\mathbb{Z}\text{,}\) \(s = \{(1, n) : n \in \mathbb{Z}\}\) and \(t= \{(n, 1) : n \in \mathbb{Z}\}\text{,}\) what are \(st\) and \(ts\text{?}\) Hint: Even when a relation involves infinite sets, you can often get insights into them by drawing partial graphs.
3.
Let \(A\) be the set of strings of 0’s and 1’s of length 3 or less. This includes the empty string, \(\lambda\text{,}\) which is the only string of length zero.
  1. Define the relation of \(w\) on \(A\) by \(x w y\) if \(x\) has the same number of 1’s as \(y\text{.}\) For example, \(01 w 100\text{,}\) but \(01 w 101\) is false. Draw a digraph for this relation.
  2. Do the same for the relation \(p\) defined by \(x p y\) if \(x\) is a prefix of \(y\text{.}\) For example, \(10 p 101\text{,}\) but \(01 p 101\) is false.
4.
Consider logical implication, \(\Rightarrow\text{,}\) on the set of propositions \(\{0,1,p,q,p\lor q, p\land q, p\land p \}\text{.}\) Draw a digraph of this relation.
